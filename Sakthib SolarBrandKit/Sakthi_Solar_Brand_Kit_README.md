# Sakthi Solar Systems — Branding Kit (v1.0)

## Included files
- `Sakthi_Solar_Brand_Kit_v1.pdf` — designer-facing brand guide
- `sakthi_logo_transparent.png` — full lockup (transparent)
- `sakthi_icon.png` — icon mark (transparent)
- `sakthi_wordmark.png` — wordmark (transparent)
- `sakthi-solar-design-tokens.json` — colors/typography/layout tokens for UI design/dev

## Core colors
- Navy: #182954
- Red:  #BB1F2C
- White: #FFFFFF

## Quick rules
- Use white space heavily (premium look).
- Navy for structure + trust; Red only for highlights/CTAs.
- Avoid adding shadows/gradients to the logo.
