## üì• Quick Download Steps

1. Visit any of the free stock sites above
2. Search for "solar panels" or "rooftop solar"
3. Download images in highest resolution available
4. Rename files to match the names above
5. Place in the correct folders (videos/ or images/projects/)
6. Commit and push to GitHub

## üöÄ After Adding Media

Once you've added the files, run:
```bash
git add public/videos public/images
git commit -m "Add solar panel images and videos"
git push origin main
```

Vercel will automatically rebuild your site with the new media!

## üí° Tips

- **Videos:** Keep under 20MB for fast loading. Compress if needed using HandBrake or online tools
- **Images:** Use TinyPNG or similar to compress images without losing quality
- **Kerala-specific:** Try searching for "tropical rooftop solar" to find images similar to Kerala climate
- **Consistency:** Choose images with similar lighting and color tones for a cohesive look
